# Project Plan: 2COMS Corporate Intranet Platform ("The Pulse")

## 1. Project Overview
A unified, AI-first corporate intranet designed to strengthen organizational culture, alignment, and engagement. The platform serves as a digital "social glue," breaking silos through high visibility and peer recognition.

## 2. Platform Scope
*   **Web Console (Full Experience):** Primary hub for Employees and HR/Admins.
*   **Mobile App (Essential Pulse):** Lightweight, engagement-focused experience for on-the-go updates.
*   **Admin Command Center:** Integrated into the Web Console for content moderation and leadership messaging.

## 3. Core Feature Pillars
### A. Alignment & Vision (Management)
*   CEO/Leadership messages & Vision boards.
*   Interactive Company Newsroom & Project Wins.
*   Engagement Calendar with visualized milestones.

### B. Silo Breaker (Networking)
*   Colleague Discovery (Skills-based directory).
*   Department Footprints (Cross-functional impact visibility).
*   Multi-tenant visibility controls for department isolation.

### C. Recognition & Culture (Social)
*   Peer Appreciation Wall & Shout-outs.
*   New Joinee Welcome Carousels.
*   Gamification: Engagement Leaderboards & Achievement Badges.

### D. Knowledge Hub (Reference)
*   Policy Vault (Handbooks, Docs).
*   Office Media Gallery (Photos/Videos).
*   Community Forum for two-way exchange.

## 4. Technical Stack
*   **Web:** React 18+, TypeScript, Tailwind CSS v4.
*   **Mobile:** React Native (Expo), TypeScript.
*   **UI Components:** Shadcn/UI (Radix), Lucide Icons.
*   **State Management:** React Context API (Modular & Lightweight).
*   **Data Layer:** Factory Pattern with Simulated Mock Services.

## 5. Architectural Standards
*   **SOLID Principles:** Ensuring maintainability and clean segregation of duties.
*   **Design Pattern:** Atomic Design (Atoms -> Molecules -> Organisms).
*   **Project Structure:** Feature-Sliced Design (FSD) for modularity.
*   **AI-First Workflow:** 80% AI generation with 20% surgical manual fine-tuning.

## 6. Execution Roadmap
*   **Phase 1:** Foundation, Shared Data Schema, and Brand Identity.
*   **Phase 2:** Web Console Development (Employee Dashboard & Features).
*   **Phase 3:** Admin Command Center (Moderation & Content Management).
*   **Phase 4:** Mobile App Development (Limited Essential Features).
*   **Phase 5:** Deployment (Vercel), Seed Data injection, and Final Polish.

## 7. Submission Deliverables
*   GitHub Repository with structured commit history.
*   `prompts.txt` documenting the iterative prompt engineering process.
*   Publicly hosted Web Console URL.
*   Video walkthrough of features, UX, and technical architecture.
